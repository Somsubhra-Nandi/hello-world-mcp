# Hello World MCP Server

A minimal implementation of a [Model Context Protocol (MCP)](https://modelcontextprotocol.io) server using TypeScript and Node.js.

This project serves as a starter template for building local MCP tools that integrate with Claude Desktop in TypeScript.

## Features

* **Type-Safe:** Built with TypeScript and Zod for robust schema validation.
* **Standard Transport:** Uses `StdioServerTransport` for seamless local integration.
* **Simple Tooling:** Includes a `say_hello` tool to demonstrate tool definition and execution.

## Prerequisites

* [Node.js](https://nodejs.org/) (v16 or higher)
* [Claude Desktop App](https://claude.ai/download)

## Quick Start

### 1. Clone and Install

```bash
git clone https://github.com/Somsubhra-Nandi/hello-world-mcp.git
cd hello-world-mcp
npm install
```

### 2. Build the Project

MCP servers run on JavaScript, so we must compile the TypeScript code first.

```bash
npm run build
```

This command compiles `src/index.ts` into `dist/index.js`.

## Configuration

To use this server with Claude Desktop, add the following configuration to your `claude_desktop_config.json` file.

**File Location:**

* **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`
* **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`

**Configuration:**

```json
{
  "mcpServers": {
    "hello-world-mcp": {
      "command": "node",
      "args": [
        "C:\\Users\\YOUR_USERNAME\\path\\to\\hello-world-mcp\\dist\\index.js"
      ]
    }
  }
}
```

> Make sure to replace `YOUR_USERNAME` and the path with your actual absolute path.
